## Install 
```sh
> git clone https://github.com/naufalardhani/facerecognition-siamesenetwork
> cd facerecognition-siamesenetwork
> pip install -r requirements.txt
> mv /path/to/{download_location}/siamese_model.h5 .
> python app.py
```
